# Support for WashU Sim Intelligence

Thank you for using **WashU Sim Intelligence**! We want to ensure you have the best experience possible while using this platform for clinical simulation and safety tracking.

## ❓ How to Get Help

### 1. Documentation
Before opening an issue, please check our comprehensive documentation:
- [README.md](./README.md): Overview and Quickstart.
- [DEPLOYMENT.md](./DEPLOYMENT.md): Detailed Cloudflare setup instructions.
- [CONTRIBUTING.md](./CONTRIBUTING.md): Guidelines for development and local setup.

### 2. GitHub Issues
For bugs, feature requests, or technical questions, please open an issue in the [GitHub Issues](https://github.com/salthepal/WashUSimIntelligence/issues) section. 
- Use the **Bug Report** template for technical problems.
- Use the **Feature Request** template for new ideas.
- For general questions, please use the **Discussion** tab if enabled, or open a regular issue.

### 3. Security Vulnerabilities
If you discover a security vulnerability, please do **not** open a public issue. Instead, follow the instructions in our [SECURITY.md](./SECURITY.md) to report it privately.

## 🤝 Community & Contributions
This is an open-source project licensed under **GPLv3**. We welcome contributions from the community! If you're interested in helping improve the platform, please see our [CONTRIBUTING.md](./CONTRIBUTING.md).
